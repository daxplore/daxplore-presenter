Main functionality
Explorer
Explore all the data. Which charts are used to visualize the data depends on settings made in Daxplore Producer. Full window width embed is recommended.
explorer.html
Profile charts
Explore several variables from a specific perspective. Supports smaller embed iframes and works well with several iframes in the same page.
Experimental functionality
Specialized visualizations that have been implemented for specific research projects. These visualizations won't be useful for most data sets. They may also not be supported by Daxplore Producer and can require manual generation of .json data files to work.
Radargraphs
An alternative to profile charts with radar charts, with specific layouts designed for the structure of data generated by the COPSOQ questionnaire.
Online example: COPSOQ Radargraph
